搜索

<DEMO>

| 参数        | 说明                 | 类型     | 默认值         |
| ----------- | -------------------- | -------- | -------------- |
| prefixCls   | 组件 class 前缀      | string   | Yep-search-bar |
| style       | 组件样式             | object   | {}             |
| className   | 组件额外样式         | string   | -              |
| value       | 输入框的值           | string   | -              |
| type        | 输入框的值类型       | string   | -              |
| onChange    | 输入改变后的回调     | function | -              |
| disabled    | 是否禁用             | bool     | -              |
| placeholder | 同 input placeholder | string   | -              |
| onSearch    | 搜索触发后的回调     | function | -              |
