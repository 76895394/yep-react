---
category: Components
title: 索引列表
---

此组件常用于 “通讯录”/“城市列表” 等场景中，支持索引导航功能。

<DEMO>

| 参数      | 说明            | 类型   | 默认值           |
| --------- | --------------- | ------ | ---------------- |
| prefixCls | 组件 class 前缀 | string | Yep-indexed-list |
| style     | 组件样式        | object | {}               |
| className | 组件额外样式    | string | -                |
