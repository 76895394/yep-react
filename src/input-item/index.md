输入框，可以带清空按钮

<DEMO>

| 参数        | 说明                               | 类型                          | 默认值                      |
| ----------- | ---------------------------------- | ----------------------------- | --------------------------- |
| prefixCls   | 组件 class 前缀                    | string                        | Yep-clearab-clearable-input |
| style       | 组件样式                           | object                        | {}                          |
| className   | 组件额外样式                       | string                        | -                           |
| value       | 输入框的值                         | string                        | -                           |
| onChange    | 输入改变后的回调                   | string                        | -                           |
| size        | 输入框大小，除默认外可选值：sm、lg | PropTypes.oneOf(['sm', 'lg']) | -                           |
| disabled    | 是否禁用                           | bool                          | -                           |
| placeholder | 同 input placeholder               | string                        | -                           |
