<DEMO>

| 参数        | 说明                                     | 类型     | 默认值 |
| ----------- | ---------------------------------------- | -------- | ------ |
| files       | 图片对象数组，每个对象必须包含 url 属性  | array    | []     |
| size        | 图片大小，需要全单位，比如"80px", "2rem" | string   | 75px   |
| addImage    | 新增图片后的处理函数                     | function | ()=>{} |
| removeImage | 删除图片后的处理函数                     | function | ()=>{} |
